# mbMouseFlags

## 语法

``` cpp
typedef enum {
  MB_LBUTTON  = 0x01,
  MB_RBUTTON  = 0x02,
  MB_SHIFT    = 0x04,
  MB_CONTROL  = 0x08,
  MB_MBUTTON  = 0x10,
} mbMouseFlags;
```

  略

- **`MB_RBUTTON`**

  略

- **`MB_SHIFT`**

  略

- **`MB_CONTROL`**

  略

- **`MB_MBUTTON`**
