# mbSetCookie

!> cookie格式必须是：Set-cookie: PRODUCTINFO=webxpress;domain=.fidelity.com;path=/;secure

## 语法

``` cpp
void mbSetCookie(
  mbWebView   webView,
  const utf8* url,
  const utf8* cookie,
);
```

## 参数

- **`webView`**

  类型：**mbWebView**

  略

- **`url`**

  类型：**const utf8\***

  略

- **`cookie`**

  类型：**const utf8\***

  略

## 返回值

类型：**void**
